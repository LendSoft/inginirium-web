

## Что не так с оригинальным сайтом

- Нет поиска по курсам — 40+ позиций, найти что-то конкретное неудобно
- Форма записи спрятана далеко, нет быстрого способа оставить заявку
- Дизайн устарел, нет анимаций, сайт выглядит статично
- У каждого курса нет своей страницы с программой и расписанием
- Нет отзывов встроенных прямо на сайте

## Анимации

- **Canvas в Hero** — сетка точек с линиями
- **Слайдер** — автопрокрутка каждые 5 секунд
- **Счётчики** — цифры считаются от 0 до значения когда блок появляется в экране
- **Scroll reveal** — элементы появляются при скролле
- **Фильтр курсов** — при смене фильтра карточки исчезают и появляются с задержкой
- **Карточки** — при наведении поднимаются, иконка немного увеличивается и поворачивается
- **Хедер** — тень появляется при скролле вниз
- **Бургер-меню** — иконка анимируется в крестик
